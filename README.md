# instant_portfolio

## setup
### for Docker
```
docker-compose build
docker-compose up -d
```

### for application
Where the application will be deployed to: localhost:3000

```
const host = process.env.HOST || '0.0.0.0'
const port = process.env.PORT || 3000
```
